# 12-exercises-for-beginners-python

Here are some beginner-level Python exercises:

### Exercise 1: Print "Hello, World!"
Write a Python program that prints "Hello, World!".

### Exercise 2: Sum of Two Numbers
Write a program that asks the user to input two numbers and prints their sum.

### Exercise 3: Check Even or Odd
Write a program that prompts the user to enter a number and checks if it is even or odd.

### Exercise 4: Calculate Factorial
Write a program to calculate the factorial of a positive integer entered by the user.

### Exercise 5: Reverse a String
Write a program that asks the user to input a string and prints that string in reverse order.

### Exercise 6: Find the Largest Number in a List
Write a program to find the largest number in a list of numbers provided by the user.

### Exercise 7: Check for Prime Number
Write a program that checks whether a number entered by the user is a prime number.

### Exercise 8: Count Characters in a String
Write a program that asks the user to input a string and counts the number of characters in that string (excluding spaces).

### Exercise 9: Print Multiplication Table
Write a program that prints the multiplication table from 1 to 10.

### Exercise 10: Calculate Area of a Circle
Write a program that asks the user to input the radius of a circle and calculates its area (A = πr²).

### Exercise 11: Create a List and Find the Sum
Write a program that creates a list of integers and calculates the sum of all numbers in the list.

### Exercise 12: Grade Classification
Write a program that asks the user to enter a score (0-100) and classifies it into grades:
- A: 90-100
- B: 80-89
- C: 70-79
- D: 60-69
- F: Below 60

Feel free to choose a few exercises to practice or combine several to create a small project. Happy coding!
